import { CustomizedSnackbars } from './snackbar';

Snackbars provide brief notifications. The component is also known as a toast.

### Component usage

```js live
<CustomizedSnackbars
    isOpen={true}
    onClose={() => setOpen(false)}
    variant={'success'}
    message={'This is a message!'}
/>
```
